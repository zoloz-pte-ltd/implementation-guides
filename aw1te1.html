# Face Compare API

# Overview

API Name: ZOLOZ SaaS Face Compare API

API URL: /api/v1/zoloz/facecompare/compare

API Description: Providing the capability of similarity comparison between two given faces, along with the face liveness detection results.

# Usage of OpenAPI SDK

To make it easy for users to access the ZOLOZ SaaS API, the OpenAPI SDK is provided. 


Firstly, users need to initialize the OpenAPI client:

// initialize OpenApiClient
OpenApiClient client = new OpenApiClient();  // data is signed and encrypted by default    
client.setHostUrl(hostUrl);
client.setClientId(clientId);
client.setMerchantPrivateKey(merchantPrivateKey);
client.setOpenApiPublicKey(openApiPublicKey);
//client.setSigned(false);     // the signature signing could be turned off
//client.setEncrypted(false);  // the encryption could be turned off

The OpenApiClient parameters:

  • hostUrl: the server's base address the merchant is going to connect to;
  • clientId: the clientId that ZOLOZ assigned to merchant;
  • merchantPrivateKey: the RSA2048 private key generated by merchant;
  • openApiPublicKey: the RSA2048 public key generated by ZOLOZ;
  • signed: true by default, merchant can set it to false;
  • encrypted: true by default, merchant can set it to false;


Secondly, users need to initialize the FaceCompareAPI handler:

// initialize FaceCompareApi
FaceCompareAPI faceCompareApi = new FaceCompareAPI(client);


Lastly, prepare the input parameters and invoke the compare method:

// prepare request body
FaceCompareRequest request = new FaceCompareRequest();
request.setBizId("test-biz-id");
request.getFace1().setContent("/9j/4AA..[omitted]..PxA=");
request.getFace1().setRect("112,234,455,89");
request.getFace2().setContent("/9j/4AA..[omitted]..PxA=");
request.getFace2().setRect("543,544,112,56");
FaceCompareResponse response = faceCompareApi.compare(request);
System.out.println(JSON.toJSONString(response));
